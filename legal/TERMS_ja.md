最終更新日: 2026-05-18
発効日: 2026-05-18

本規約は、Chrome 拡張機能「tab-coach」(以下「本拡張機能」) の利用条件を、提供者と利用者の間で定めるものです。利用者は、本拡張機能をインストールまたは利用した時点で、本規約のすべての条項に同意したものとみなされます。同意できない場合は、直ちに本拡張機能のインストールを中止し、既にインストール済みの場合はアンインストールしてください。

## 第2条 (定義)

本規約において、以下の用語は次の意味を有します。

| 用語 | 定義 |
|------|------|
| 本拡張機能 | tabisurushosai が公開する Chrome 拡張機能「tab-coach」、およびそのソースコード・ドキュメント・関連配布物の総称 |
| 利用者 | 本拡張機能を Chrome ブラウザにインストールまたは利用するすべての自然人および法人 |
| Premium 機能 | 第4条第2項に定める、将来提供される可能性のある有償オプション機能 |
| 個人データ | 個人を識別できる情報および GDPR/CCPA 等で個人データとして定義されるもの |
| 整理 (Cleanup) | 本拡張機能が未アクティブ・重複・読了タブを `chrome.tabs.remove` で閉じる動作 |
| アーカイブ (Archive) | 整理対象タブの URL・タイトル・日時を `chrome.storage.local` に保存する機能 |
| スナップショット (Snapshot) | 整理直前のタブ状態を保持し、30 秒以内のアンドゥを可能にするローカル記録 |
| ホワイトリスト (Whitelist) | 利用者が指定する、整理対象から除外する URL パターンの集合 |
| Manifest V3 | Chrome 拡張機能の最新仕様。本拡張機能は本仕様に準拠 |
