# プライバシーポリシー (tab-coach)

最終更新日: 2026-05-17

## 1. 収集する情報

tab-coach は **一切の個人情報を収集しません**。

外部サーバーへの通信は行わず、アクセス解析・トラッキング・広告配信ツールも導入していません。

## 2. ローカル保存データ

以下のデータは、利用者のブラウザ内 (`chrome.storage.local`) にのみ保存されます。
ネットワーク経由で外部に送信されることはありません。

- ユーザー設定 (バッジ閾値、未アクティブ判定分数、ダークモード等)
- ホワイトリスト (整理対象外にする URL パターン)
- アーカイブ済タブ情報 (タイトル・URL・ファビコン URL・保存日時)
- 整理実行履歴 (件数・カテゴリ・日時)
- 直近の整理スナップショット (アンドゥ用、30秒で消去)
- 読了判定キャッシュ (URL と読了時刻の対応)

これらのデータは、本拡張機能のアンインストール時に Chrome により削除されます。

## 3. 拡張機能の権限

本拡張機能は、以下の最小限の権限のみを利用します。

- `activeTab`: アクティブタブの操作 (整理時)
- `tabs`: タブ一覧の取得・更新・削除 (機能の中核)
- `storage`: 上記ローカルデータの保存

ホスト権限 (`host_permissions`) は要求しません。

## 4. 第三者への提供

本拡張機能は、収集したデータがないため、第三者にデータを提供することはありません。

## 5. 子どもの個人情報

本拡張機能は、子どもを含むすべての利用者から個人情報を収集しません。

## 6. ポリシーの変更

本ポリシーを変更する場合は、本ファイルおよび GitHub リポジトリの CHANGELOG にて告知します。
