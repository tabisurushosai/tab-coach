## 詳細 (Detailed description)

Chrome Web Store の「詳細」フィールド (16,000 文字以内)。マークダウンは無効、改行のみ反映されます。アップロード時は本セクションを丸ごとプレーンテキストで貼り付けます。

```
タブの開きすぎを可視化し、ワンクリックで整理する Chrome 拡張機能です。完全オフラインで動作し、外部サーバーへの送信・広告・トラッキングは一切ありません。

【こんな方におすすめ】
・気付くとタブを 30 件以上開いてしまう
・「あとで読む」のタブが溜まり続けている
・同じサイトを誤って複数回開いてしまう
・ブラウザの動作が重く感じる
・タブを閉じるかどうか毎回迷ってしまう

【主な機能】
1. タブ数バッジ
ツールバーアイコンに現在のタブ数を表示します。10 件未満は灰色、10〜19 件は黄色、20 件以上は赤色で警告。閾値はオプション画面でカスタマイズできます。

2. ワンクリック整理
ポップアップから「未アクティブ」「重複」「読了」「すべて」のカテゴリでタブを絞り込み、該当タブを一括で閉じられます。

3. 30 秒アンドゥ
整理直後の 30 秒間はアンドゥボタンが表示され、閉じたタブをまとめて復元できます。誤操作してもすぐに取り戻せます。

4. アーカイブ
タブを「閉じる」のではなく「アーカイブ」することもできます。アーカイブしたタブは後からオプション画面で復元したり、JSON でエクスポート・インポートしたりできます。

5. ホワイトリスト
URL パターン (例: https://*.github.com/*) を登録すると、整理の対象から除外できます。

6. 月次レポート
これまでに整理したタブの件数と、推定節約時間 (1 タブあたり 15 秒換算) を月別に表示します。

7. アクセシビリティ
ダークモード (自動 / ライト / ダーク)、文字サイズ調整 (80〜150%)、ハイコントラスト表示、キーボードショートカット (Ctrl/Cmd+Shift+T でポップアップ、Ctrl/Cmd+Shift+R で整理) に対応。

【プライバシー】
・外部サーバーへの送信は一切ありません (完全オフライン)
・ユーザーの個人情報・閲覧履歴・アクセスログを収集しません
・データは chrome.storage.local にのみ保存され、同期もしません
・アカウント登録・ログイン不要
・広告・トラッキング・分析タグ一切なし
・収益化のための販売・第三者提供は一切しません

【必要な権限】
・tabs: タブ一覧・タイトル・URL を取得して分類するため
・storage: 設定・アーカイブ・履歴をローカルに保存するため

これら以外の権限 (host_permissions、識別子、リモートコード読み込み等) は一切要求しません。

【動作環境】
・Chrome 114 以上
・Manifest V3 準拠
・日本語 / English 自動切替 (Chrome の言語設定に追従)
```
